# UI/UX Design Checklist
**Team:** GTRs Testers  
**Reviewer:** DvChege  
**Application:** Bookstore App  
**Date:** November 2025  

This checklist is used to evaluate the visual consistency, usability, and navigational experience of the Bookstore Application.

| UI Area | Check | Status (Pass/Fail/Pending) | Notes |
|--------|-------|---------------------------|-------|
| Layout | Content is aligned and evenly spaced | Pass |  |
| Buttons | Buttons are clearly labeled and clickable | Pass |  |
| Text | Font sizes are readable and consistent | Pass | Clear and Visible to the user |
| Navigation | User can move between pages easily | Pass | The navigation is smooth |
| Colors | Colors are consistent across all pages |pass |  |
| Images | Book images render clearly and are not stretched | Pass | The sizing of the Images is perfect |
| Forms | Form fields are properly labeled and validated | Pass | Form is clear and properly visible |
| Error Messages | Errors are descriptive and easy to understand | Pass | When user fails to input where required the message is clear |
| Responsiveness | Works on both mobile and desktop screens | Pass | Using Android |
| Accessibility | Can be used with keyboard only (no mouse required) |Pass  | Using tab key in the form  |
